# Meeting Protocol

Скилл и вспомогательные материалы для автоматизации создания протоколов встреч через OpenCode.

## Назначение

Агент на основе транскрипции встречи создаёт оформленный HTML-протокол в Confluence, а затем рассылает action points ответственным через Outlook.

## Содержимое

| Файл | Описание |
|------|-------------|
| [meeting-protocol.md](./meeting-protocol.md) | Скилл агента: логика, шаги, режимы, HTML-шаблоны |
| [corrections.json](./corrections.json) | База правок: фамилии, названия команд, термины для автоподстановки |

## Режимы работы

| Режим | Описание |
|-------|-------------|
| **Добавить** | Создаёт новый протокол на основе транскрипции. Просит куда сохранить и какой тип протокола |
| **Редактировать** | Просит ссылку на существующий протокол в Confluence, затем уточняет что изменить |

## Принципы работы

- **Один вопрос — один ответ**: агент ждёт ответа перед тем как задать следующий вопрос
- **Не додумывай**: факты, сроки и ответственных берутся только из транскрипции
- **Подтверждение перед отправкой**: письма уходят только после явного оказа пользователя
- **Автоподстановка**: агент использует `corrections.json` для подстановки верных фамилий и названий
