# Rocket.Chat Thread Summary — Инструкция по установке скилла

Скилл позволяет вашему ИИ-агенту забирать тред Rocket.Chat по ссылке и формировать структурированное саммари (выводы, открытые вопросы, сроки).

## Совместимость

- **Платформа:** opencode
- **Язык:** русский (интерфейс скилла и вывод саммари)

## Установка

1. Скопируйте папку `rocket-chat-thread-summary/` в директорию скиллов вашего агента:
   ```
   ~/.config/opencode/skills/rocket-chat-thread-summary/
   ```

2. Агент автоматически обнаружит скилл по метаданным из `SKILL.md`.

## Настройка

Файл конфигурации не требуется. При первом запуске агент запросит учётные данные Rocket.Chat.

### Получение учётных данных

Скилл использует аутентификацию через Personal Access Token или через `authToken`/`userId`, полученные через Rocket.Chat API.

#### Способ 1: Personal Access Token (рекомендуется)

1. В Rocket.Chat откройте **Account** → **Personal Access Tokens**
2. Создайте новый токен, скопируйте значение
3. При запросе агента передайте токен и свой User ID

#### Способ 2: Через браузерные cookies

1. Откройте Rocket.Chat в браузере
2. Откройте DevTools (F12) → вкладка **Application** → **Cookies**
3. Найдите `rc_token` (это `authToken`) и `rc_uid` (это `userId`)
4. Скопируйте их, когда агент попросит

Агент использует эти данные через переменные окружения `ROCKET_CHAT_AUTH_TOKEN` и `ROCKET_CHAT_USER_ID`.

## Использование

Отправьте агенту один из этих запросов:

- `/rocket-chat-thread-summary`
- "сделай саммари треда"
- "summary треде"
- "выводы из треда"
- "подведи итоги треда"

Агент попросит ссылку на тред в формате:
```
https://rc.alfa-bank.net/direct/{room_id}?msg={thread_id}
```

## Что делает скилл

1. Аутентифицируется в Rocket.Chat через REST API
2. Загружает все сообщения треда (до 200)
3. Формирует структурированное саммари:
   - Список участников (полные имена)
   - Описание темы
   - Главный и краткий итог обсуждения
   - Открытые вопросы
   - Упомянутые сроки
   - Итог одной фразой

## Ограничения

- **Только чтение** — никогда не пишет в Rocket.Chat
- **Только факты** — выводы делаются строго на основе содержимого треда
- **UTF-8** — корректно обрабатывает кириллицу через запись в файл (не через `print()`)
- **Привязан к workspace** — `rocket_chat_url` в SKILL.md установлен как `https://rc.alfa-bank.net`; измените, если ваш инстанс другой
- **Токен истекает** — агент перезапросит учётные данные при получении 401

## Кастомизация под ваш Rocket.Chat инстанс

Отредактируйте строку 11 в `SKILL.md`:
```yaml
rocket_chat_url: https://ваш-rc-инстанс.com
```

Также обновите формат ссылки в описании скилла (шаг 2 в `SKILL.md`).
